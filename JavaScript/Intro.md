# Introdução a programação Javascript
- Histórico

## JavaScript
 Padrão para definir comportamento de documentos HTML, adcionando interatividade ao documento, acessando todos os elementos e textos do documento

 ### Exemplos de Interatividade
 - Validação de formulários
 - Carregamento dinâmico de conteúdo
 - Menus de navegação responsivos
 - Slider e carroséis de imagens
 - Atualização de conteúdo em tempo real
 - Animações efeitos visuais
 - Integração de mapas e geolocalização

 ### Js no Documento

 #### Scripts Incorporados
 - O código dendro do doc <script>..</script>
 - Alterando apenas o documento
 - Pode ser dentro de um elemento <head> ou no final <body>

 ### Scripts externos
 - Documentos separado contendo o código 
 - Arquivo texto com extensão .js
 - Importado com elemento <script> com atributo src=".."
 - Varios documentos HTML podem usar o mesmo script
